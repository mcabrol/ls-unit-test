# ls-unit-test

**Simple bash script for verify our implementation of ls command.**

## Configure

Configure binary path:
```
FTLS=/path/to/ft_ls
```

Set log output path:
```
LOG=/path/log/output/
```

# Run test

Start the shell script:
```
./run.sh
```

For enable the leaks check:
```
./run.sh -leak
```

**For leaks testing, Valgrind required.**
