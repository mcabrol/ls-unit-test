**Simple bash script for verify our implementation of ls command.**

Configure exe/log paths:
```
FTLS=/path/to/ft_ls
LOG=/path/to/log
```

For disable the leaks check:
```
sh ls_check.sh disable-leak
```
**For leaks testing, Valgrind required.**
